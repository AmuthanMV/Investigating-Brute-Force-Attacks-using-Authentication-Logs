# Investigating-Brute-Force-Attacks-using-Authentication-Logs
A beginner-friendly cyber security project to detect brute force attacks using authentication logs.


## 📌 Project Overview
This project focuses on detecting brute force attacks by analyzing authentication log files. A brute force attack is a type of cyber attack where an attacker repeatedly tries different username and password combinations to gain unauthorized access to a system.

This project was developed as a beginner-friendly cyber security mini project using Python. It reads authentication logs, identifies repeated failed login attempts, detects suspicious IP addresses, and highlights successful logins after multiple failures.

---

## 🎯 Objective
The main objective of this project is to investigate suspicious login activities and identify possible brute force attack patterns using authentication logs.

---

## 🚀 Features
- Reads and analyzes authentication log files
- Detects repeated failed login attempts
- Identifies suspicious IP addresses
- Finds successful logins after multiple failed attempts
- Displays most targeted usernames
- Generates a simple security investigation report
- Beginner-friendly and easy to understand
